Claude Cowork Prep Guide

Goal: Arrive with Claude Desktop installed, Cowork available, and a clean workshop folder ready for sample workshop files.

What You Need

  • A paid Claude plan (Pro, Max, Team, or Enterprise)
  • The latest Claude Desktop app installed on your laptop
  • Permissions to manage available Claude features (if currently blocked)

5-Minute Setup

  1. Install or update Claude Desktop from claude.com/download
  2. Open the app and sign in
  3. Confirm you can see the Cowork tab near the top of the app
  4. Open Cowork, then click Customize in the left sidebar
  5. Confirm the Customize area opens and shows Customize Skills, Browse Plugins, and other related settings
  6. Create a local folder for workshop materials (e.g., vcpeit-cowork-workshop)

If Something Is Missing

  • If the Cowork tab does not appear: you may need a supported paid plan, an admin enablement step on Team or Enterprise, or a current Claude Desktop version
  • If Customize is blocked or incomplete, check with your internal Claude admin
  • If your organization controls Skills separately, ask your admin to confirm code execution and file creation are enabled

Good To Know

  • Cowork runs in the desktop app, not the web app
  • Keep Claude Desktop open while Cowork is working
  • Sample materials will be used in the session — no need to preload real firm data
